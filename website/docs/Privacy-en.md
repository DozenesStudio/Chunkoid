## 1. Introduction

Chunkoid (hereinafter referred to as "the Software") is an Android native Minecraft world converter developed by Dozener. The Software's core conversion engine utilizes the open-source Chunker-cli (MIT License) from The Hive Games company (hereinafter referred to as "The Hive"). The Software developer, Dozener, solely provides the Android application packaging, user interface, and auxiliary feature development, and bears no development responsibility for the core conversion engine's code logic.

We highly value user privacy and are committed to protecting your personal data security.

This Software is completely open-source and free. You can view the complete source code on [GitHub](https://github.com/DozenesStudio/Chunkoid) to verify the authenticity of this privacy policy. The source code for the core conversion engine, Chunker-cli, is available on [GitHub](https://github.com/HiveGamesOSS/Chunker).

## 2. Data Collection

This Software **does not collect, store, or transmit any user personal data**.

- This Software does not collect your name, email, phone number, geographic location, or other personal information.
- This Software does not collect your device information, usage logs, network behavior, or similar data.
- This Software does not upload your Minecraft save files to any server.
- This Software does not automatically report crash logs or error reports through any third-party frameworks or system services.

## 3. Data Processing

All data processing is completed locally on your device:

- Reading, converting, and writing operations for Minecraft save files are all performed locally.
- Log files generated during the conversion process are stored only locally on your device.
- Converted save files are saved only to the local directory you specify.

## 5. Permissions

This Software requires the following permissions to function properly:

- **Storage Permission**: Used to read and write Minecraft save files (Android 11+ uses scoped storage mechanism).
