### Basic Operation Instructions

1. Open the app and click "Select World"
2. Select the world file to convert (supports .zip / .mcworld / folder formats)
3. The app automatically finds worlds within packages, no repeated extraction needed
4. Select target platform and version
5. Click "Start Conversion"
6. Wait for conversion to complete
7. View or export converted worlds in the Output Manager

### Settings

- **JVM Max Memory**: Increase memory allocation when converting large worlds
- **Preserve Original NBT**: Keep original NBT data in worlds
- **Theme Switch**: Choose preferred wallpaper theme in settings
- **Output Management**: Clear all output worlds
- **Developer Options**: Show unreleased features
